MyResume, built with react.js

## Getting Started

### Clone Repository

`git clone`

### Install Dependencies

`pnpm install`

### Bundle Files

`pnpm build`

### Display it on local browser

`pnpm start`

### Edit files with Live Reloading

`pnpm watch`
